# CASIA TCAS - Programme

Environnement Python 3.13 pour le forecasting de séries temporelles (Scikit-learn, XGBoost, LSTM/TensorFlow).

## Installation

```bash
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Dépendances

- numpy, pandas
- scikit-learn (Random Forest, TimeSeriesSplit)
- xgboost
- tensorflow (LSTM/Keras)
